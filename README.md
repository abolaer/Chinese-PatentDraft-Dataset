# Chinese-PatentDraft-Dataset
Chinese-PatentDraft-Dataset

数据集聚焦于新一代信息产业技术领域，涵盖人工智能、云计算、大数据、物联网、区块链、5G通信、集成电路等多个方向，具有较强的技术代表性与实际应用价值。数据集面向专利文本生成任务统一整理而成，针对不同来源文本在表达方式、信息组织和写作粒度上的差异，本研究结合专利文书生成需求进行了筛选和规范化处理，形成可用于模型训练与评测的底稿式文本。
在字段组织上，本研究以单篇专利为基本样本单位，对各类文本进行统一对齐与整理，最终形成包含专利底稿、专利标题、专利摘要和权利要求书等核心字段的数据样本。本研究采用分层抽样方法，按照8:1:1的比例将数据集划分为训练集（20745条）、验证集（2593条）和测试集（2593条）。数据划分以专利样本为基本单位进行，确保同一专利对应的信息不会同时分布在不同子集中，避免数据泄漏对实验结果造成干扰。本数据集规模可观、构建过程严谨，为专利生成任务提供了坚实可靠的数据支撑。

考虑到知识产权等原因，我们暂时公开经脱敏处理的部分数据集，以便其他研究者参考与复现相关实验，后续我们会逐步增大公开数据集的规模
